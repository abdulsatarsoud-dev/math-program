# Python Math Programs

A collection of beginner Python programs for basic math calculations. Built to practice logic and conditionals 

## Program Included

1. **Quadratic Equation Solver**:
   
   Solves equations of the form ax<sup>2</sup> + bx + c = 0. Calculate the real roots , discriminants , sum and product of roots.
   
3. **Profit and Loss Program**:
   
   Calculate profit, loss, and profit/loss percentage based on cost price and selling price.
   
4. **Area and Circumference of Circle**:
   
   Computes the area and circumference of a circle given the radius. Use &pi; for accuracy.
   
6. **Interest Program**:
   
   Calculates simple interest based on principal, rate and time.
